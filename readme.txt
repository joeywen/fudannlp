FudanNLP主要是为中文自然语言处理而开发的工具包，也包含为实现这些任务的机器学习算法和数据集。
本工具包及其包含数据集使用LGPL3.0许可证。
使用方法请参考http://code.google.com/p/fudannlp

开始配置：
1. 编码设为UTF8
2. 将lib下所有jar包加入classpath

文件组织结构：
/src 		基本功能代码
/lab		实验中代码
/test		Junit单元测试代码
/apps	应用代码
/webservice	WebService代码
/example 	对外API使用样例
/example-data 	对外API使用样例数据
/models	模型文件
/scripts		执行脚本


使用样例参考example目录下代码。

命令行调用方式：
http://code.google.com/p/fudannlp/wiki/fudannlp_cli
API调用方式：
http://code.google.com/p/fudannlp/wiki/fudannlp_api
